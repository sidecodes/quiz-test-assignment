# Quiz Assignment

This application uses Angular5 for the front-end framework  with segregated modules according to views and components that are being utilised inside the view. It helps break down to chunks of view specific data which makes independent components to talks to each other for data flexibily. 

The idea is to register user which he uses for login, attends test and can view his result. Thats how the test application has been made.

# To run the project

Clone and npm install. After that ng serve.
